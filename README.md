# Aroda-je

## Installation
Cloner le dépôt

Ouvrez votre terminal et exécutez la commande suivante :
```
git clone https://github.com/Ximawa/Arosa-je.git
```

Accéder au répertoire du projet
```
cd arosa-je
```

Installer les dépendances
```
npm install
```

Lancement de l'application
```
npx expo start
```
